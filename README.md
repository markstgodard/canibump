# CAN I BUMP ?

## About

canibump is a simple command-line tool to check if
[canibump website](https://canibump.cfapps.io) is **YES** or **NO**.

## Install

    go get github.com/markstgodard/canibump

## Usage

    $ canibump
    Can I bump status is: YES
